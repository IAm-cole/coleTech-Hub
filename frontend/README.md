





# Project Name
MERN["MongoDB", "EXPRESS", "REACTJS" "NODEJS"] LOGIN/REGISTER


## Description

This projects is a full stack project showing full backend of user login, register and storing each data on Mongodb and writing a strong authentication and error managment, bycrpt is used for password hashing to secure our password credentials from attackers, jsonWebToken to generate token to each user to remember their login data for future login.
