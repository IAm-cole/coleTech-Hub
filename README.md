# ⚡ ColeTech-Hub (A clean, fast, and user-friendly e-commerce platform designed for buying and selling electronics.) 

## 📝 About the Project
This project is a fully functional online electronics store. It allows users to browse a wide variety of gadgets, manage a shopping cart, and securely check out. The platform is built to handle high-resolution product images, keep track of inventory, and provide a smooth shopping experience from homepage to payment.

## ✨ Core Features
- **Product Catalog**: Easily browse, search, and filter electronics by category (e.g., phones, laptops, Speakers).
- **Shopping Cart**: Dynamic system to add, remove, and update item quantities before buying.
- **User Accounts**: Secure login and registration to track orders and save shipping addresses.
- **Admin Dashboard**: A private panel for store owners to add new products, update prices, and manage stock levels.
- **Secure Checkout**: Simple and safe payment processing for a realistic shopping simulation.

 ## Available Commands

This project is built using **Vite** and **React**, which provides a blazing-fast development environment. 

To run the application locally, you need to start both the frontend and the backend server.

### 1. Start the Frontend
Run the development server:
```bash
npm run dev
```
Once started, open [http://localhost:5173](http://localhost:5173) in your browser to view the application. The page will automatically reload whenever you save changes to your code.

## 🛠️ Built With

### Frontend
- **React** - Component-based user interface library
- **Vite** - Modern, ultra-fast frontend build tool
- **Tailwind CSS** - Utility-first framework for modern styling
- **Axios** - Promise-based HTTP client for API requests
- **Lucide React** - Clean and consistent tech icon library
- **Swiper** - Touch-responsive slider for product showcases

### Backend & Authentication
- **Node.js** - JavaScript runtime environment
- **Express** - Fast, minimalist web framework for the API
- **JSONWebToken (JWT)** - Secure user authentication and session tokens
- **BcryptJS** - Password hashing for secure user accounts

### Database & Tools
- **MongoDB** - NoSQL database to store electronic products and user data
- **NPM** - Package manager to manage project dependencies
